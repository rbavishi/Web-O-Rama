  Median Algorithm for Disk-resident Data   

Sept '14 - Nov '14

  • Independently developed a 2-pass deterministic and a 2-pass randomized algorithm to find median of large data-sets (1 Terabyte) along with rigorous performance tests on a set of self-made benchmarks

Deterministic Algorithm -

  • Final results on the deterministic algorithm were found to be very close to the actual paper by Munro-Paterson (1980)
  • An ε-approximate median (ε = 1/ n) is calculated in the first pass, with the help of which, the exact median is computed in the second pass. The ε error-range ensures efficient computation of the median

Randomized Algorithm -

  • Basic Random-Sampling techniques were implemented using Mersenne-Twister PRNG. Theoretical success probability was calculated and compared with the practical performance over thousands of program-runs
  • Success probability close to 0.6 achieved for medium-sized data-sets (10-50 GB)



  P2P Dropbox   

Sept '13 - Nov '13

  • A Linux application for back-up and syncing of files between two or more peers
  • Users have a shared folder across different machines, with local copies, in which any changes made are synced across all devices
  • Linux inotify API was used to track changes in the shared folder in order to initiate syncing
  • rsync was used to sync changes in files/folders so as to ensure efficient transfer of files. Only the changes between local and remote copies are transferred
  • Multithreading with mutexes was used to parallelize syncing operations
  • Retry mechanisms and network detection system were implemented to ensure syncing even in networks with inter-mittent connectivity
  • Implemented a Command-Line-Interface to add/delete folders and show synchronization status
